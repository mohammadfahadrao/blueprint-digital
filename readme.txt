Plugin to accompany form creation from API response.

== Description ==

BluePrint-Digital-Forms uses BluePrint-Digital API to create and process forms on wordpress websites.
After activating the plugin, add the shortcode `[blue_print_form]` to render the form in any page of 
your WordPress website.
